Compensating transactions as a service protocol defines a RESTful transaction model for compensating transactions using HTTP protocol semantics such as entity tags, conditional requests, hypermedia controls, and media types.

Introduction

This documents describes the compensating transactions as a service protocol and provides examples for the protocol interactions.

Notes

Internet Media Type

The Internet Media Type / MIME Type for Transaction resources is "application/vnd.transaction+turtle".

The Internet Media Type / MIME Type for service command resources is "application/vnd.service.action+turtle".

The character set RDF Patch is "UTF-8".
